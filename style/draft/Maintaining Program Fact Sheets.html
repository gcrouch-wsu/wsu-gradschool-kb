Fact Sheets on the Graduate School website provide application details for degree and certificate programs. They include application links, program and degree descriptions, faculty and student counts, campus locations, English proficiency test requirements when applicable, GRE and GMAT requirements when applicable, and other relevant information. Fact Sheets are linked from graduate program websites and WSU's graduate application system, so they are important recruiting tools.

Access to Fact Sheets

Graduate program coordinators and directors are assigned through myWSU Faculty Lists by checking one of the following boxes.

This assignment provides access to the Graduate School WordPress site for editing Fact Sheets. You may also request access by submitting a ticket to the Graduate School Service Desk.

Understanding Roles and Permissions

Fact Sheet editing is controlled by WordPress user roles and Team Access, also called Editorial Access Manager (EAM), settings.

Key Terms

  • Open Access: Standard WordPress permissions apply.
  • Team Access: Only assigned users or roles can edit.
  • Restricted Users: Contributors and Subscribers.
  • Restricted Fields: Fields that cannot be edited by limited users.

Role-Based Editing Summary

  • Administrators have full control. Graduate School Fact Sheet administrators are part of the Communications and Outreach team, managed by Dr. Greg Crouch. Contact the Graduate School if restricted fields need to be reviewed or edited.
  • Editors and Authors can edit Fact Sheets, but they must be assigned if Team Access is enabled.
  • Contributors have limited editing ability. Most department coordinators will be given Contributor access.
  • Subscribers can edit only when they are explicitly assigned and given direct access.

Using Team Access (EAM)

Team Access restricts editing at the individual Fact Sheet level.

  • Off: Open access.
  • Roles: Only selected roles can edit.
  • Users: Only selected individuals can edit.

Only administrators can update Team Access settings.

Restricted Fields

Certain fields cannot be edited by restricted users, including the Fact Sheet title, program name, degree type, URLs, locations, and Team Access settings.

If roles need to be changed for new employees or duty changes, contact the Graduate School Outreach and Technology team through the Graduate School Jira Help Desk.

Editing Fact Sheets

Go to the Graduate School WordPress admin site and log in with your WSU NetID.

In the dashboard, select Factsheets from the left menu.

The Factsheets screen lists Fact Sheets for programs across Washington State University. You will have access only to the Fact Sheets relevant to your department. Fact Sheets you can edit have blue, clickable titles.

To locate your Fact Sheet, use Search Factsheets in the upper-right corner. Mouse over the Fact Sheet title to display the edit link.

Alternative Method for Accessing Fact Sheets

In the upper-left corner of the dashboard, select The Graduate School to open the Graduate School website.

From the Graduate School homepage, use the left navigation to select Graduate Degree Programs. This page lists all program Fact Sheets in alphabetical order. You can find a Fact Sheet by using the search feature or selecting a letter.

When you find the Fact Sheet you want to edit, select Edit Factsheet. If you do not have access to that Fact Sheet, the Edit Factsheet link will not appear.

The edit screen opens for the selected Fact Sheet. Do not edit the Fact Sheet title because changing it can disrupt the link. Program and plan degree name change requests must go through the Office of the Provost and WSU Faculty Senate.

Fact Sheet Fields

The following fields are visible when editing Fact Sheets.

  • Factsheet display name (Restricted): The display name shown for the Fact Sheet.
  • Accepting Applications (Required): This field may be adjusted based on recruiting goals.
  • Include in programs list (Restricted): If this option is set to No, the Fact Sheet remains available through its direct URL but will not appear in the programs list.
  • Total grad students (Required): Enter the program's approximate number of graduate students. This number may be updated at any time.
  • Total Graduate Faculty (Required): Enter the number of graduate program faculty.
  • Total Core Graduate Faculty (Optional): Some graduate programs list a subset of core faculty, especially when representing a specialization or plan under a program. For example, the School of Biological Sciences has two Ph.D. programs: Doctor of Philosophy in Biology and Doctor of Philosophy in Plant Biology. For the Ph.D. in Plant Biology, the core faculty number is smaller. If this field is not used, it will not be visible.
  • Aided grad students (Optional): Some programs list the number of students on appointment as a recruiting tool. If this field is not used, it will not be visible.
  • Program Handbook URL (Required): Provide the link to the graduate student handbook.
  • Student Learning Outcomes (Required): WSU's regional accreditor, Northwest Commission on Colleges and Universities (NWCCU), requires graduate program and plan student learning outcomes to be public. These outcomes may be listed in this field, or the field may include a link to a program website. This field may not be blank.
  • Application URL (Restricted): The application link for the Fact Sheet.
  • Degree home page (Required): Provide a link to the graduate program website. This link should be the intended landing page for potential graduate students.
  • Locations (Restricted): Graduate degrees are approved for specific campuses. A college dean must seek approval from the provost and Faculty Senate to extend an existing degree to a new campus.
  • Global Campus URL (if different from physical campus): If a program also has a Global Campus program, enter the URL in this field.
  • Priority deadlines (Required): List priority deadlines for graduate applications. Enter the specific deadline in box A and other details in box B.
  • Language Test Requirements (Optional): See WSU Graduate School international student admission requirements. Students who satisfy option A, B, or C are exempt from English proficiency exams. Applicants who do not meet one of these options must earn minimum scores on an accepted English proficiency exam, such as TOEFL or IELTS. Graduate programs may require higher scores than the Graduate School minimum. If a program has higher score requirements, list those program-specific scores here. They will not be visible if this field is not used. Place the score in box C, the test name in box D, and other details in box E.
  • Additional Program Requirements (Optional): Admissions exams such as the Graduate Record Examination (GRE) or Graduate Management Admission Test (GMAT) are not required by the Graduate School but may be required by individual programs. If a program has additional requirements, list them here. Place the test name in box F and choose whether it is required in selection box G. If this field is not used, it will not be visible.
  • Contact Information (Required): Specify the program's main contact point or contact information. Place the graduate coordinator or director name in box H and the contact email in box I. You may add multiple entries using Add and remove entries using Remove. The Graduate School recommends using a shared mailbox rather than an individual email address. For example, sbs.gradstudies@wsu.edu is a shared mailbox for graduate studies in the School of Biological Sciences. Shared mailboxes may be requested through the Information Technology Services Managed Account Request.
  • Description of the graduate degree (Optional): Briefly describe the degree program.
  • Admission Requirements (Optional): List any program-specific application requirements beyond the materials students submit when applying to the Graduate School.
  • Student Opportunities (Optional): Describe opportunities such as professional development, internships, travel, or other relevant experiences.
  • Career Opportunities (Optional): Describe common post-graduation career opportunities.
  • Career Placements (Optional): Describe graduate placement outcomes.

You may preview your changes before publishing by selecting Preview Changes in the upper-right corner of the Fact Sheet. The preview opens in a new tab. When editing is complete, select Update to publish.
