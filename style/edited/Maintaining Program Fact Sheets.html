Start here: What you need to know

This page is for graduate program coordinators, program directors, and department staff who maintain program Fact Sheets on the Graduate School website. Prospective students read Fact Sheets but do not edit them; the information you maintain here feeds graduate program websites and WSU's graduate application system.

Fact Sheets provide application details for degree and certificate programs: application links, program and degree descriptions, faculty and student counts, campus locations, English proficiency requirements, admission exam requirements, and other recruiting information. This page covers how to get access, how roles and permissions work, how to open and edit a Fact Sheet, and what each field needs.

Before you begin: get access

Graduate program coordinators and directors receive Fact Sheet access when they are assigned in myWSU Faculty Lists by checking the graduate program coordinator or graduate program director box.

This assignment provides access to the Graduate School WordPress site for editing Fact Sheets. You may also request access by submitting a ticket to the Graduate School Service Desk.

Roles and permissions

Fact Sheet editing is controlled by WordPress user roles and Team Access, also called Editorial Access Manager (EAM), settings.

Key terms

  • Open Access: Standard WordPress permissions apply.
  • Team Access: Only assigned users or roles can edit.
  • Restricted Users: Contributors and Subscribers.
  • Restricted Fields: Fields that cannot be edited by limited users.

What each role can edit

  • Administrators have full control. Graduate School Fact Sheet administrators are part of the Communications and Outreach team. Contact the Graduate School if restricted fields need to be reviewed or edited.
  • Editors and Authors can edit Fact Sheets, but they must be assigned if Team Access is enabled.
  • Contributors have limited editing ability. All graduate program coordinators and directors have Contributor access. Contributors cannot create new Fact Sheets or edit restricted fields.
  • Subscribers can edit only when they are explicitly assigned and given direct access.

Team Access (EAM) settings

Team Access restricts editing at the individual Fact Sheet level.

  • Off: Open access.
  • Roles: Only selected roles can edit.
  • Users: Only selected individuals can edit.

Only administrators can update Team Access settings.

Restricted fields

Certain fields are locked for restricted users and appear greyed out on the edit screen: the Fact Sheet title and web address, Factsheet display name, Include in programs list, Application URL, Global Campus URL, Locations, the Program Names and Degree Types selections, and Team Access settings. These fields are marked Restricted in the field tables below.

You do not need to avoid the locked fields — WordPress prevents changes to them. Program and degree name changes must go through the Office of the Provost and WSU Faculty Senate; contact the Graduate School if a Fact Sheet title or display name needs updating.

If roles need to change for new employees or duty changes, contact the Graduate School Communications and Outreach team through the Graduate School Service Desk.

Edit a Fact Sheet

There are two ways to open a Fact Sheet for editing: from the WordPress dashboard or from the Graduate School website.

Method 1: Open from the WordPress dashboard

  1. Go to the Graduate School WordPress admin site and log in with your WSU NetID.
  2. In the dashboard, select Factsheets from the left menu.
  3. Find your Fact Sheet with Search Factsheets in the upper-right corner.
  4. Hover over the Fact Sheet title and select Edit.

The Factsheets screen lists Fact Sheets for programs across Washington State University, but you can open only those relevant to your department. Fact Sheets you can edit have blue, clickable titles.

Method 2: Open from the Graduate School website

  1. In the upper-left corner of the dashboard, select The Graduate School to open the Graduate School website.
  2. In the left navigation, select Graduate Degree Programs. The page lists all program Fact Sheets in alphabetical order.
  3. Find your Fact Sheet with the search feature or by selecting a letter.
  4. Select Edit Factsheet. If you do not have access to that Fact Sheet, the link does not appear.

The edit screen opens for the selected Fact Sheet. The Fact Sheet title and other restricted fields are locked and appear greyed out (see Restricted fields above).

Fact Sheet fields

The tables below list each Fact Sheet field, whether it is required, and what to enter. Fields marked Restricted appear greyed out and can be changed only by Graduate School administrators — contact the Graduate School to update them. Optional fields are hidden on the published Fact Sheet when left blank.

Why learning outcomes are required: WSU's regional accreditor, the Northwest Commission on Colleges and Universities (NWCCU), requires graduate program and plan student learning outcomes to be public. The Fact Sheet links to the page that lists them.

Core faculty example: The School of Biological Sciences offers a Ph.D. in Biology and a Ph.D. in Plant Biology. The Plant Biology Fact Sheet lists a smaller core faculty count than the program-wide faculty total.

Priority DeadlinesOptionalAn earlier deadline for funding consideration. Choose the semester (Fall, Spring, or Summer), enter the deadline, and add the international deadline when it differs. Use Add and Remove for multiple entries.

Locations: Graduate degrees are approved for specific campuses. A college dean must seek approval from the provost and Faculty Senate to extend an existing degree to a new campus.

Priority vs. Program Deadlines: Program Deadlines are the dates after which applications are no longer reviewed. Priority Deadlines are earlier, optional dates that tell applicants when to apply to be considered for funding. If your program treats them the same, enter only the Program Deadline.

Language test background: See the Graduate School's international student admission requirements. Applicants who meet one of the exemption options on that page do not take an English proficiency exam; all other applicants must earn minimum scores on an accepted exam, such as TOEFL or IELTS. Enter scores here only when your program requires more than the Graduate School minimum.

Publish your changes

  1. Select Preview Changes in the upper-right corner of the Fact Sheet. The preview opens in a new tab.
  2. When the preview looks right, select Update.

Update publishes immediately. Your changes are live on the Graduate School website as soon as you select Update.

Need help?

If you have questions or need help editing your Fact Sheet, submit a ticket to the Graduate School Service Desk.
